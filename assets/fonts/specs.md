# Font Asset Specs

## User-facing behavior

- The landing page typography is backed by the local font files in this directory.

## Implementation rules

- Store `Ioskeley` as `Ioskeley.woff2`.
- Store `jikitourai.regular` as `jikitourai.regular.ttf`.

## Constraints and invariants

- Keep both font files local and reference them from `index.html`.
